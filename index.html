마통 업체 크롤러 — 개인정보처리방침

최종 수정일: 2026년 2월 13일

1. 개요

"마통 업체 크롤러" 크롬 확장프로그램(이하 "본 확장프로그램")은 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 본 확장프로그램이 어떤 데이터를 수집하고, 어떻게 사용하며, 어떻게 보호하는지에 대해 설명합니다.

2. 수집하는 정보

본 확장프로그램은 사용자의 개인정보를 수집하지 않습니다.

본 확장프로그램이 처리하는 데이터는 다음과 같습니다:

- 마통(msgtong.co.kr) 사이트에 공개된 마사지 업체 정보 (업체명, 주소, 카테고리, 가격, 평점, 리뷰 수 등)
- 사용자가 설정한 크롤링 옵션 (선택 지역, 딜레이 값, 재시도 횟수 등)

이 데이터는 모두 사용자의 브라우저 로컬 스토리지(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 데이터 사용 목적

수집된 업체 데이터는 사용자가 엑셀 파일로 다운로드하여 업무 목적으로 활용할 수 있도록 하는 데에만 사용됩니다.

4. 데이터 저장 및 보안

- 모든 데이터는 사용자의 로컬 브라우저에만 저장됩니다.
- 외부 서버, 제3자 서비스, 클라우드 등으로 데이터가 전송되지 않습니다.
- 사용자는 브라우저의 확장프로그램 데이터를 삭제하거나, 확장프로그램을 제거하여 언제든지 저장된 데이터를 삭제할 수 있습니다.

5. 제3자 공유

본 확장프로그램은 어떠한 데이터도 제3자와 공유하지 않습니다.

6. 쿠키 및 추적

본 확장프로그램은 쿠키를 설정하지 않으며, 사용자의 브라우징 활동을 추적하지 않습니다.

7. 아동 개인정보

본 확장프로그램은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
